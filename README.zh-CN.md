# X 互关与关注助手

一个本地优先的工具，用于清理和管理你的 X / Twitter 关注与粉丝列表——找出没有回关你的
账号、加白名单、按批次取关或回关。完全在浏览器中运行：无后端、无需登录、数据不出本机。

[English](README.md) | 简体中文

## 两种使用方式

| Track | 目录 | 适合 |
|---|---|---|
| **油猴脚本**（Tampermonkey） | [`userscript/`](userscript/) | 已经在用 Tampermonkey——一键安装 + 自动更新 |
| **Chrome 扩展**（独立） | [`extension/`](extension/) | 不想装 Tampermonkey——直接加载进 Chrome |

两者共用同一套逻辑与界面（扩展的 `content.js` 由 userscript 生成）。选其一即可——
同时启用只会出现两个面板。

## 功能

面板会出现在 x.com / twitter.com 页面上，提供两个工作流：

### 1. 清理未互关
- 扫描你的「正在关注」页面（自动下滑加载全部）。
- 通过 X 显示的「关注了你」标签判断是否互关。
- 只列出**没有回关你**的账号。
- 可把想保留的账号加入白名单，白名单内的账号绝不会被取关。
- 勾选账号，设置本次数量与间隔，批量取关。

### 2. 关注 / 回关
- 扫描任意账号的「所有关注者」或「认证关注者」页面——你自己的（回关）或别人的（直接关注）。
- 列出可以关注的账号。
- 勾选账号，使用相同的数量 / 间隔控制批量关注。

### 白名单与导出
- 白名单支持以 JSON 或每行一个 `@用户名` 的形式导入 / 导出。
- 未互关列表可导出为 JSON。

## 使用

1. 输入用户名（你自己的或别人的），打开「正在关注 / 所有关注者 / 认证关注者」页面。
2. 点击「扫描」，等待自动滚动到底部。
3. 用「查看未互关 / 查看认证回关 / 查看全部回关」切换列表。
4. 勾选账号（或「全选」），设置「本次数量」和「间隔毫秒」，再点「开始取关 / 开始回关」。
   「停止」可中断正在进行的批处理。

提示：
- 「本次数量」限制每次处理的账号数；「间隔毫秒」是每次操作之间的等待时间——建议保持较大值
  （默认 4500 毫秒）以避免触发限流。
- 结果列表约占面板一半高度并可独立滚动。

## 安全

- 取关和关注只有在你勾选账号并点击「开始取关 / 开始回关」后才会真正执行。
- 建议先把「本次数量」设为 **1**，确认在你的账号上行为正常。
- 所有状态（白名单、勾选、进度）只保存在浏览器的 `localStorage` 中。

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)。
2. 打开脚本地址：
   https://raw.githubusercontent.com/jackdaidai/x-follow-cleaner/main/userscript/x-follow-cleaner.user.js
3. 在 Tampermonkey 中点击「安装」。
4. 打开任意 X 页面，右上角会出现面板。

## 说明

- 不想用 Tampermonkey？改用[独立 Chrome 扩展](../extension/)。
- 自动更新通过 jsDelivr（脚本头部的 `@updateURL`）从本仓库的 `userscript/` 路径拉取。
  如果你之前从旧的根目录地址安装过，请用上面的链接重新安装一次——移动后旧路径已失效。
